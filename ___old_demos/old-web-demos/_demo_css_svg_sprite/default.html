CSS 3 SVG Sprite Demo

In this demo, we are showcasing using SVG files instead of PNG, GIF or JPG files for images, specifically as a 'sprite' - using one SVG file, for mutliple images across several HTML elements.
This sample uses CSS :hover to move the background-position-y value, instead of changing standard CSS attributes like background-color.
